# 안녕하세요, 비즈니스 가치를 창출하는 풀스택 개발자 김형년입니다!

<br>

안녕하세요! 기술로 세상의 문제를 해결하고, 사용자 중심의 서비스를 만들어가고자 하는 풀스택 개발자 김형년입니다.  
**기획(PM)부터 아키텍처 설계, 인프라 구축, 배포 및 실제 서비스 운영**까지 전 과정을 주도하는 'Full-Cycle 개발자'의 역량을 갖추고 있습니다.

---

## 🌟 핵심 역량 (Key Competencies)

**Product Development**  
기획부터 배포까지 제품의 전체 라이프사이클(Full Lifecycle) 리드 가능  

**Scalable Backend**  
Java 21 및 Spring Boot 기반의 견고하고 확장성 있는 서버 아키텍처 설계  

**User-Centered Frontend**  
React와 TypeScript를 활용한 직관적이고 반응성이 뛰어난 UI/UX 구현

## 🚀 서비스 운영 및 프로젝트 (Services & Projects)

### 🎓 [AllTeachers](https://allteachers.co.kr) (Active)
**B2B 동영상 LMS(학습 관리 시스템) 플랫폼**  
1인 총괄 개발로서 비즈니스 요구사항 분석, 아키텍처 설계, 백엔드/프론트엔드 구현, 클라우드 인프라 구축 및 배포를 완료했습니다.  
현재 안정적으로 서비스를 운영하며 사용자 피드백을 기반으로 지속적인 개선 작업을 수행하고 있습니다.  

<br>

### 🎯 [JobCatchers](https://github.com/BroYears/jobcatchers) (In Progress)
**B2B 멘토·멘티 매칭 및 인턴십 시스템 플랫폼**  
전체 프로젝트 리딩 및 DB 아키텍처 설계를 주도하며, 대규모 매칭 알고리즘 구현을 위한 백엔드 시스템 개발에 집중하고 있습니다.
